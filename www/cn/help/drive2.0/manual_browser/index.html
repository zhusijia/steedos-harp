一、移动到兄弟文件夹下

删除文件夹

如果在某文件夹下，想删除子文件夹，则移动鼠标到子文件夹这一行最右侧的删除图标，

这时，删除图标变为红色，并出现“删除”的提示框，点击删除图标即删除文件夹。

删除后的文件夹 还会显示在左下角链接“已删除文件”中，点击可进入查看和恢复。180天后，系统会自动永久删除他们。在此之前，您如果要手动永久删除，点击右侧的删除图标

这时，删除图标变为红色，并出现“永久删除”的提示框，点击“永久删除”即永久删除文件夹。

如果要恢复，则点击“恢复”图标即可。

删除（注意不是“永久删除”）后的文件夹和文件，保留在“已删除文件”中。180天后，系统会自动执行永久删除。

尽管如此，限于每个用户的空间限制，系统仍然可能在180天之前永久删除这些文件。

原则是保留的已删除文件的大小不能超过可使用空间大小的50% 。

一旦保留的已删除文件的大小达到可使用空间大小的50%，系统将自动启动永久删除，从最早的版本文件开始永久删除，直至满足保留的已删除文件的大小不超过可使用空间大小的50%为止。

文件操作

文件上传

一、点击上传

点击导航条后面的上传图标  ，弹出选择文件的页面，选择后点击“打开”即可，如果不上传，点击“取消”。

二、拖动上传

将文件管理器（Windows系统下的）或Finder（Mac下的）的一个或多个文件拖进 相应文件夹 即可。

文件版本

一、生成新版本

如果上传的新文件与文件夹中已经存在的文件重名，则系统弹出冲突处理页面如下：

（1）如果勾选新文件

则新上传的文件为这个文件的最新版本，而原来的文件则成为历史版本。

这时，点击文件名后的“版本”图标，则出现此文件的历史版本。

（2）如果勾选已经存在的文件，则不上传新的文件。

（3）如果同时勾选了两个文件，则新上传的文件名会加一个数字，这样，文件夹中的文件就是两个，而不是一个文件的两个版本。

二、恢复旧版本

如果要恢复某个旧版本，则点击文件名称后的“版本”图标，

出现此文件的所有历史版本，点击需要恢复的版本后面的“恢复”图标，则此版本成为最新版本，而原来的最新版本成为历史版本。

三、版本保留机制

上传同名文件时，系统会生成新的版本。受限制于用户的空间大小，每次生成新版本时，系统会作自动检查来删除某些旧版本，以控制空间：

  • 第1秒时，保持一个版本；
  • 前10秒，每2秒保持一个版本；
  • 前1分钟，每10秒保持一个版本；
  • 前1小时，每1分钟保持一个版本；
  • 前24小时，每1小时保持一个版本；
  • 前30天，每1天保持一个版本；
  • 30天后，每1个星期保持一个版本。

此外，版本控制时还会注意，增加新版本时不会使用超过50%的用户目前可用空间。如果存储的版本超过此限制，会自动删除旧版本直到满足空间限制。

使用WebDAV访问华炎云盘

WebDAV（Web-based Distributed Authoring and Versioning）是一种基于 HTTP 1.1协议的通信协议，在GET、POST、HEAD等几个HTTP标准方法以外添加了一些新的方法，使应用程序 可直接对Web Server直接读写，并支持写文件锁定(Locking)及解锁(Unlock)，还可以支持文件的版本控制 。

华炎云盘同时提供了WebDAV服务，使得您可以在Microsoft Windows、Mac OSX等操作系统中，使用WebDAV来访问华炎云盘里的文件，和浏览器访问实现同样的功能，如上传、下载、移动等。

Microsoft Windows

通过资源管理器访问华炎云盘，请作如下操作：

1.打开资源管理器

2.选择“映射网络驱动器”

3.选择驱动器（例如Y盘），录入文件夹为华炎云盘WebDAV的服务器地址：

https://drive.steedos.com/remote.php/webdav/

4.点击“完成”，输入用户名、密码，点击“确定”按钮。

5.资源管理器中，增加Y盘（drive.steedos.com）

6.点击即可进入华炎云盘系统，您可以在这里查看、增加、移动、删除文件夹，也可以查看、增加、移动、删除文件，这些操作与在Web页面的操作是一致和同步的。

使用同步客户端

华炎云盘为Windows系统、Mac OS系统提供了同步客户端，通过同步客户端，您可以：

  • 指定一个或多个本地的文件夹，
  • 与华炎云盘上的文件，保持实时同步。

安装客户端

您可以在骏马云平台官网的以下网页上下载各操作系统的同步客户端，http://www.steedos.com/cn/drive/downloads/

下载后双击运行,直至唇线配置页面，其中连接的服务器地址为：https://drive.steedos.com

然后，还要输入您的用户名、密码

还有就是选择本地的文件夹路径

其中，“本地文件夹”，点击进入即可修改将华炎云盘同步到本地哪个文件夹。

“选择同步的文件”就是选择只同步华炎云盘的哪几个文件夹还是同步所有的文件夹。

勾选需要同步的文件夹，点击“OK”回到刚才的页面。

点击“连接”按钮完成文件夹配置，

您可以分别点击进入浏览器访问华炎云盘，也可以点击进入查看本地文件夹。点击“完成”即完成安装和配置。

使用

安装完成并配置好用户后，您就可以在资源管理器中对华炎云盘中的您的文件进行操作，由客户端进行实时同步，使得本地和服务器上的文档保持一致。

您可以进行以下操作：

  • 增加、删除、移动、修改文件夹
  • 增加、删除、移动、修改文件

同时，在系统的状态栏上会有华炎云盘的图标，表示客户端同步的不同状态。图标显示样子会有以下不同的几种：

  表示已连接到服务器

  表示正在同步

  表示同步被暂停

  表示未连接到服务器

  表示有消息

  表示有错误，例如连接服务器的URL有误，或是用户名密码有误等

右键此图标，出现如下菜单

  • 点击“在浏览器中打开”，就可以在浏览器中打开华炎云盘
  • 点击“打开文件夹”，就可以打开本地文件夹，执行相应操作。

下面是已使用空间的提示。

点击“最近修改”可看到最近修改的记录。

点击“设置”可进行账户、动态、常规、网络等的设置。
